# Student Performance Dashboard Analysis

This project analyzes student performance and study behavior using Python.

## Overview
The project explores how study habits and background factors affect academic performance through multiple dashboards.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
